# Menu-Website
# ☕ A simple cafe menu website built using HTML and CSS.

## 🌟 Features
* 📱 Responsive design
* 🧭 Navigation menu
* 🍹 Drinks, 🍔 Foods, 🍿 Snacks, and 🍎 Fruits sections

## 🛠 Technologies Used
* HTML5
* CSS3


In the summer, during one-month I was trying to learn HTML and CSS. One idea came to my mind that, I can create a mini website, which can be related to mini cafe or restaurants. So, I decided to work, while watching tutorials and doing a lot of practices. Of course, during the journey, AI was helping me too. Instead of AI doing my job, I was clearly focused on learning and correcting my mistakes and making some creative adjustments.

This is my first work. I can say, I am really proud that in such a short time, I was able to do and show my work.
Looking forward to the challenges and works
